# 🔢 Scientific Calculator

A modern scientific calculator built using **React + Vite**.  
This project provides a clean and responsive interface for performing advanced mathematical calculations.

## 🚀 Features

✔ Basic arithmetic operations  
✔ Scientific functions  
✔ Dark mode interface  
✔ Memory functions  
✔ Calculation history  
✔ Responsive modern UI  

## 🛠 Technologies Used

- React.js
- Vite
- JavaScript
- HTML5
- CSS3
